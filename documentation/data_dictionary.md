\# Data Dictionary - Multilingual Support Analytics



\## Dataset Overview

\- \*\*Total Records:\*\* 210+ support tickets

\- \*\*Date Range:\*\* November 2023 - March 2025

\- \*\*Languages:\*\* English, Spanish, French, German, Portuguese

\- \*\*Regions:\*\* Europe (97%), Americas (3%)



\## Key Fields



| Field Name | Type | Description | Example Values |

|------------|------|-------------|----------------|

| Ticket\_ID | String | Unique identifier | TKT-001 |

| Customer\_Language | String | Customer's language | en, es, fr, de, pt |

| Agent\_Language | String | Agent's language | en, es, fr, de, pt |

| Satisfaction | Integer | CSAT score (1-5) | 1=Low, 5=High |

| Resolution\_Time | Float | Hours to resolve | 2.5, 4.8 |

| Priority | Integer | Urgency level | 1=Low, 3=Critical |

| Region | String | Geographic area | Europe, Americas |

| Issue\_Type | String | Support category | Software, Hardware, Other |



\## Key Metrics



\- \*\*Average Satisfaction with Language Match:\*\* 4.5/5.0

\- \*\*Average Satisfaction without Match:\*\* 3.4/5.0

\- \*\*Impact:\*\* 31% satisfaction increase

\- \*\*Resolution Time Improvement:\*\* 30% faster with match
